#SHL GenAI Assessment Recommendation System

This project implements a Generative AI–based retrieval system to recommend
relevant SHL assessments for a given hiring query or job description.

# Approach
- Sentence-BERT embeddings for semantic understanding
- Cosine similarity–based retrieval
- Top-K SHL assessment URL recommendation

# Evaluation
The system is evaluated using Mean Recall@K on the provided labeled dataset,
as specified in the SHL assignment.

# Deployment
- Streamlit web application for live demo
- Public GitHub repository for reproducibility
